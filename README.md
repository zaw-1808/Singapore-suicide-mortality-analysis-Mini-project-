# Singapore Suicide Mortality Analysis (2015–2024)
## Project Overview
This project explores suicide mortality trends in Singapore from 2015 to 2024 using publicly available government data. The goal is to analyze patterns across demographic subgroups and visualize trends using Python.
The analysis includes data cleaning, transformation, and interactive visualization to better understand changes in suicide mortality rates over time.
This mini project is part of a data analysis learning portfolio focused on public health data.
## Objectives
Explore suicide mortality trends over time
Compare mortality rates across demographic subgroups
Practice data cleaning and transformation using Python
Create interactive visualizations for data exploration
## Dataset
The dataset contains annual mortality rates attributed to suicide and other major diseases in Singapore.
Key variables used in this project include:
Variable	Description
Year	Year of recorded data
MainCategory	Main demographic category
Subcategory	Subgroup within the category
Rate	Mortality rate per 100,000 population
## Data Source
Data retrieved from the Singapore Government Open Data Portal.
Dataset: Mortality Rates Attributed To Suicide, Cardiovascular Disease, Cancer, Diabetes Or Chronic Respiratory Disease (Annual)
Agency: Singapore Department of Statistics (SINGSTAT)
